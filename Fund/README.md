# 基金趋势预测工具 v2.0

一款基于 Windows 环境的轻量级桌面工具，实现"一键运行"自动抓取真实历史基金数据，利用多种算法模型预测未来一个月趋势，并生成可视化图表及操作建议。

## 功能特性

- **数据获取**：自动从 AkShare 获取基金历史净值、规模等数据，支持多基金代码输入
- **市场特征**：自动获取沪深300、中证500指数及国债收益率等多维特征
- **数据清洗**：自动处理缺失值、异常值，统一数据格式
- **多模型预测**：支持 6 种预测模型，预测未来 30 个交易日走势
- **图表生成**：自动生成趋势图、预测对比图、资金流向图，支持导出 PNG/JPG
- **策略建议**：基于预测结果给出"买入/卖出/持有"建议及理由
- **一键启动**：双击即可运行，自动检测并安装依赖

## 预测模型

| 模型 | 类型 | 特点 | 推荐场景 |
|------|------|------|---------|
| **集成 (Ensemble)** | 加权集成 | 组合多模型结果，精度最高最稳定 | **默认推荐** |
| **Prophet** | 趋势+季节性 | 自动捕捉趋势转折点和年度周期 | 稳健型参考 |
| **ARIMA** | 经典统计 | 理论扎实，短序列稳定，可解释性强 | 数据量较少时 |
| **LSTM** | 深度学习 | 学习序列模式，对趋势变化敏感 | 数据量充足时 |
| **MultiVar-LSTM** | 多变量深度学习 | 引入市场指数特征，精度提升15-30% | 需要最高精度 |
| **MA** | 移动平均线 | 最基础，计算最快 | 快速参考 |

**模型降级机制**：当首选模型失败时，自动按优先级降级（如 LSTM → Prophet → MA），确保始终能产出预测结果。

## 安装与运行

### 方式一：直接运行（需要 Python 环境）

1. 确保已安装 Python 3.8+
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行：
   ```bash
   python -m src.main
   ```
   或双击 `run.bat`

### 方式二：打包为 EXE（无需 Python 环境）

1. 安装 PyInstaller：
   ```bash
   pip install pyinstaller
   ```
2. 执行打包：
   ```bash
   python build.py
   ```
3. 生成的 `FundPredictor.exe` 位于 `dist/FundPredictor/` 目录下

## 使用步骤

1. 启动软件后，在顶部输入框输入 6 位基金代码（如 `000001`）
2. 多只基金用逗号分隔（如 `000001,110011`）
3. 选择预测模型类型（默认"集成"推荐）
4. 点击"开始分析"按钮
5. 等待分析完成，查看：
   - **趋势预测图**：历史走势 + 未来 30 天预测曲线及置信区间
   - **回测对比图**：模型回测效果对比
   - **资金流向图**：基金规模变化趋势
   - **策略建议**：买入/卖出/持有建议及理由
6. 点击"导出 PNG"或"导出 JPG"保存图表

## 常见问题

**Q: 提示"网络连接失败"怎么办？**
A: 请检查网络连接，确保能访问互联网。AkShare 数据源需要联网使用。

**Q: 提示"基金代码不存在"怎么办？**
A: 请确认输入的是正确的 6 位数字基金代码。可在东方财富等平台查询基金代码。

**Q: Prophet 模型预测失败怎么办？**
A: 软件会自动降级为 ARIMA 或 MA 模型。Prophet 需要安装 prophet 库（`pip install prophet`）。

**Q: LSTM 模型预测失败怎么办？**
A: 软件会自动降级为 Prophet 或 MA 模型。LSTM 需要 PyTorch 库和至少 60 个交易日数据。

**Q: 集成模型为什么比单一模型更准确？**
A: 集成方法通过回测 MAPE 自动计算各模型权重，加权组合多个模型的预测结果，降低单一模型偏差风险。

**Q: 图表中文显示为方框怎么办？**
A: Windows 系统通常自带中文字体。如遇问题，请确认系统已安装"微软雅黑"或"黑体"字体。

**Q: 依赖安装失败怎么办？**
A: 请手动执行 `pip install -r requirements.txt`，或使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```
